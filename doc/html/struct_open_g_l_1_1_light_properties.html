Detailed Description

Light Properties. LightProperties struct allow us to have a reference to memory order in the uniform object buffer. Update this struct if you need more light's data.

See also
Light::UpdateUniformBlock
Warning
If you update this struct make sure to update also the Light::UpdateUniformBlock function. You have to update also the shader program uniform buffer object, the UBO layout must match the order declared in this struct.
